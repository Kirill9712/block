# Block for Revenge

Локальный чёрный список для мобильного Discord-клиента с модификацией Revenge.

## Установка

Добавьте этот адрес в **Revenge → Plugins**:

```text
https://kirill9712.github.io/block/
```

Если GitHub Pages ещё не включён, можно использовать прямой адрес:

```text
https://raw.githubusercontent.com/Kirill9712/block/main/
```

## Команды

- `/block user:` — скрыть пользователя; без аргумента показывает список.
- `/unblock user:` — вернуть пользователя.

В поле `user` можно выбрать человека или вставить Discord ID.

Плагин скрывает сообщения, ответы, индикатор печати и пользователя в списках/войсе. Голос и Soundboard скрытых пользователей локально отключаются.

Исходный код находится в [`src/index.ts`](src/index.ts).
